# Local runtime data (not committed)

This folder is used for **local runtime artifacts** generated by Sigmaris-OS.

- `metaMemory.json`: local MetaMemory log (may include message-like content) → kept out of the public repo
- `growth.json`: local growth log → kept out of the public repo
- `*.db`: local SQLite files (dev-only) → kept out of the public repo

Tracked templates:
- `metaMemory.example.json`
- `growth.example.json`
